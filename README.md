# Minty Tools
![done](https://img.shields.io/badge/Project-Passing-green)

A tools package for Termux.
# Installation

1) `pkg install git`

2) `git clone https://github.com/krishtechstudio/mintytools`

3) `cd mintytools`

4) `chmod +x *`

5) `./requirements.sh`

6) `./run.sh`
